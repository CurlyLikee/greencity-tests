# GreenCity - Events Page Test Cases

## Description

This repository contains manually written test cases for the **Events** page of the GreenCity web application.  
The goal is to practice writing structured test cases using the standard format (Title, Preconditions, Test Steps).

---

## Page Under Test

**URL:** https://www.greencity.cx.ua/#/greenCity/events

---

## Repository Structure

```
greencity-tests/
├── README.md
└── test-cases/
    └── events-page-tests.md
```

## Test Cases Overview

| ID | Title | Type |
|----|-------|------|
| TC-01 | Events list is displayed on page load | Positive |
| TC-02 | Filtering events by category shows relevant results | Positive |
| TC-03 | Clicking on an event opens its detail page | Positive |
| TC-04 | Search with no matching results shows a message | Negative |
| TC-05 | Events page handles no internet connection gracefully | Negative |
